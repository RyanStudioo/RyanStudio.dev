Features

Varying Personality

Oracle will vary its personality based on how you speak to it, the message rating system is used to change oracle's personality over time

Long Term Memory

Oracle is able to recall a large amount of memories based on selected inputs by its developers, community inputs, webscraping, and more!

Image Vision

Oracle can see and react to images you send it!

Accounts system

Users can pin messages to get Oracle to remember things about them or have a customised personality
